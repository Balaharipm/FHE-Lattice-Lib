# FHE-Lattice-Lib
Objective:  The first step in mastering FHE is building a clean, well-documented library of the underlying mathematical primitives. This project serves as the engine for all subsequent FHE implementations.

Key Components

   1.  Polynomial Arithmetic: Implement efficient polynomial addition, multiplication, and modular reduction over rings.
   2.  NTT Implementation: Develop a standard Number Theoretic Transform (NTT) for fast polynomial multiplication.
   3.  Sampling Algorithms: Implement discrete Gaussian sampling and uniform sampling, which are critical for generating the error terms in LWE/RLWE problems [1].
